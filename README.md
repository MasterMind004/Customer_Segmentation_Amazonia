🧠 Problem Statement

Retail businesses often apply uniform strategies to all customers, resulting in inefficient marketing and reduced profitability. The absence of structured customer segmentation limits the ability to identify high-value customers and optimize engagement strategies.

This project addresses the problem by analyzing customer data and categorizing users based on their purchasing patterns and behavior.

🎯 Objectives
Develop a structured customer segmentation model
Identify high, medium, and low-value customers
Analyze purchasing behavior across multiple dimensions
Build an interactive dashboard for business insights
Enable data-driven decision-making
